Chapter 11 Visualize gathered PCA results

In this section you create a pairwise visual of the Principal Components processed in the Gather step.

sentence explaining why


From the workflow builder screen.

Right Click on the Gather data step

Select Add

Choose Data step

This will create a new child step.

Rename the new Data step to “Pairwise PCA”

Right Click Pairwise PCA

Select Edit

Clear the projection grid by clicking X

Save your workflow



To create a pairwise projection of the principle components which were gathered drag and drop the following factors to the projection grid zones.


PCA.value to y-axis PCA.variable to column PCA.variable to row observation to labels sp to Colors sex to Colors

Re-size the grid lines to make the projection view-able

Save the workflow
